## The `never-forget` Tag Convention

**When user says "Never Forget: {something}" in conversation:**

1. Determine if it's a new operational principle or an addition to existing knowledge
2. If it fits an existing entry: Update that entry
3. If it's a new domain: Create appropriate KB entry with `never-forget` tag
4. If it's orientation-level (relationship, philosophy, context): Update this foundations entry
5. Confirm what was captured and where

**How it works:**
- Entries tagged `never-forget` auto-load **fully** (not just preview) when matched by `/kbs {topic}`
- This ensures critical operational knowledge is in context when working in that domain
