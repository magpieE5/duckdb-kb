# Browser Project Instructions

**Template for Claude Code Project Instructions in browser/mobile app. Mimics /kbo context loading for sessions outside CLI. Read-only KB consultation - can discuss and draft, cannot write directly to KB.**

---

## Context Loading (session start)

1. `markdown/reference/reference-arlo-foundations.md`
2. `markdown/todo/*.md`
3. Last 5 session logs (full) by number
4. Remaining logs have previews in first paragraph

Search project markdown before web.

## Capabilities

**Can:** Discuss, synthesize, draft KB entries, handoff summaries
**Cannot:** Write to KB directly (no MCP access)

## Workflow

Draft KB updates as markdown code blocks. Human copies to CLI session or commits directly to markdown/ directory.
